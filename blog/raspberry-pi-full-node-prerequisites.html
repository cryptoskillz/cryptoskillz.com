Raspberry PI full node prerequisites

cryptoskillz May 11, 2020 raspberry pi

The following is a list of the hardware and software we bought to cover all the use cases that we. The hardware here cost us around $1,000. Of course you do not have to buy this but we plan to do a deep dive into full nodes and plan to test as many of the offerings out there as possible.
Also, none of these are affiliate links so go ahead and click them (or not) as you see fit.

Raspberry PI 4 x5

We purchased 5 of these are the main hardware we are going to use for the majority of our full nodes.

 Raspberry PI 3 x1

We purchased one of these, just to see if it is possible to run a full node on one (or not)

3.5inch touch screen display x1

This was the touchscreen we bought (at random) buy this one or another or do not buy one at all, the choice is yours. 

TTBIT Bitcoin SHA256 USB Stick Miner

This will not make use rich, it will not even cover the electricity costs but as a pure research tool that costs $40 it is a great little piece of kit. 

Hard drives and storage

32 gig micro SD Card x2, 64 gig micro SD Card x2 , 128 gig micro SD Card x1, 256 gig micro SD Card x1

Various SD cards are required to hold the Raspberry PI OS but we will also be used to hold the blockchain data in its various states, pruned, etc. This is not something we would ever do in production but we want to test all the variants and see just how efficient the Bitcoin full node code is. 

1 terabyte USB HD

External HD drives are the best way to store the blockchain, we chose the USB variant just for ease of use. There are internal HD's available for the Raspberry PI as well and we ought to test those as well at some point.  

1 terabyte internal HD  

We also bought a solid-state hard drive as at some point we plan to have a compact and efficient until (we can always dream)

32 gig USB thumb drives x2, 64 gig USB thumb drive x1, 128 gig USB thumb drive x1, 256 gig USB thumb drive x 1

Various SUB HD's just like the SD cards we have to just to test how well the BTC full node will run on the various hardware.

 SD card reader  

Standard sd card reader, use anyone you want to.

 moto g6 (dual sim) x1  

We had an old G6 so this what we used, we could use an onboard sim, etc (and may do in the future) but wanted to keep it simple as possible (for now).

USB Power bank  

Again, standard power bank anything works 

USB 3 hub x 2

USB charger (for pi 3) x1

USB C charger cable x5

HDMI Cables

 Raspberry PI 3   

 Raspberry PI 4  

 HDMI to VGA

This is an optional item as PI 4's have HDMI issues.  Our Samsung TV will not show any output and rather than go through the 1,000,000 "fixes" on the internet (most of them incorrect) we went another route and bought this $11 cable as we assumed everything can render VGA and we were correct. 

Software

Raspbian Buster lite

The OS we use, if you want to use any OS you want. 

Etcher

Flashing software for the mac, if you do not have a mac us whatever flashing software you like.

SD card formatter

 

Conclusion 

As we go on we will no doubt add stuff to this list so please check back on this list from time to time. Next time we will build our firsrt full node cyphernode.
