Environment Variables

We keep all of the environment in a .env file. For security reasons, this has been added to the git ignore but there is a .env_copy that contains a template you can use. You will see a file called .env_copy which contains all of the variables just rename this to .env and add the details below. 

EMAILSMTP = 
EMAILUSERNAME = 
EMAILPASSWORD =  
WALLETPASSPHRASE = ''
WALLET = 'spending01.dat'
NETWORK = 1
RPCUSERNAME = rpcusername
RPCPASSWORD = rpcpassword
RPCHOST = 127.0.0.1
RPCPORT = 8332
PORT = 3030
CONFIRMATIONS=1
STRIKEENDPOINT=                           
STRIKEAPIKEY=                                
LIGHTNETWORK=2
AUTOPAYMENT=1
AUTOPAYMENTTICK=600000

 emailsmtp

Hold the email SMTP server. For development, we are using https://ethereal.email/ which fakes an email server allowing us to test the functionality without actually sending emails.

emailusername

This is the username for the SMTP service

emailpassword

This is the email password for the SMTP service

walletpassphrase

The wallet passphrase for bitcoin core.

wallet

the wallet account we are using for bitcoin core

network

this is the testnet (1) or mainnet (2)

rpcusername

The username of the RPC server

rpcpassword

The password of the RPC server

rpchost

The ip address of the RPC server

 rpcport

The RPC port of the server.

port

The port you want to run the server on.

Confirmations

How many confirmations we wait for

strikeendpoint

The endpoint for strike https://strike.acinq.co/

strikepaikey

The API key for strike https://strike.acinq.co/

ligthning network

this is the testnet (1) or mainnet (2)

autopayment

Decide to release funds to cold storage as soon as we have enough confirmations.

autopaymentcheck

This is a timer that checks to see if there are any funds to be released.
