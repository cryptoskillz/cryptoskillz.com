BTC Full nodes with Raspberry PI part 1

cryptoskillz May 6, 2020 raspberry pi

Introduction

Hello friends, it has been a while since we wrote anything, in that time there has been a global pandemic and BTC has surged, as is to be excepted :].

Anyhoo, cryptoskillz has not been idle, they have been in deep research mode, remember that 6-month tech sabbatical that started over 2 years ago, yeah that is still ongoing.

 What have we been doing?

Glad you asked, we have been deep diving hardware full nodes and learning heaps about how that whole fun world works.  Now we have learned a decent amount we are going to document it in a level of detail that borders on being unnecessary definitely uninteresting to all but a very few specific people and we are ok with that.

What are we going to cover?

Another fine question,  we plan to cover the following topics

Basic

  • Build a bitcoin full node on a raspberry pi with a 1tb HD
  • Build a bitcoin full node on a raspberry pi with USB sticks ranging from 32-256 GB
  • Build a bitcoin full node on a raspberry pi with a 32 gig SD card

Third party

  • Build a BTC pay server on a Raspberry PI
  • Build a cyphernode server on a Raspberry PI
  • Build a raspiblitz server on a Raspberry PI
  • Build a mynode server on a Raspberry PI
  • Build a CASA node  on a Raspberry PI
  • Build a ROKOS server on a Raspberry PI
  • Build a BTC full node on a pine phone

Mining

  • Build a miner on a Raspberry PI
  • Build a miner on a Raspberry PI (with various USB miners)
  • Look into mining pools

Lightning

  • Build a lightning ATM

Testnet

Build our own testnet and have a selection of the above on it to simulate the BTC network.

Conclusion

This should keep us busy for another year or two, splendid
